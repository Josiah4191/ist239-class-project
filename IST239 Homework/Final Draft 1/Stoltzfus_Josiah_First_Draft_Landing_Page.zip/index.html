Lab 1-1

In Lab 1-1, I learned how to create a script inside of an html document, and I use the document.write() function. The function receives a string argument and can be used to output HTML content to a webpage.

Lab 1-3

In Lab 1-3, I learn how to embed JavaScript into html tags. We do this by using the on-click event handler inside of the input tag, so that when a button is clicked, another element is changed so that a message appears on the screen.

Lab 1-5

In Lab 1-5, I learn how to debug errors found in the html document. This execise tests my understanding of common errors related to syntax, case sensitivity, function use, and string formatting in JavaScript. All required changes have been completed and documented in the code comments.

Lab 2-1

In Lab 2-1, I created functions to convert values between Celsius and Fahrenheit. These functions are triggered when the user enters a temperature in either input field on the webpage.

Lab 2-3

In Lab 2-3, I used JavaScript to show messages when you hover over shapes. It helped me learn how to change page content with events.

Lab 2-5

In Lab 2-5, I fixed several syntax and logic errors in the JavaScript code. I corrected event handlers, fixed the condition that was preventing the 0 button from working.

Lab 3-1

In Lab 3-1, I learned how to work with collections of HTML elements using class names in JavaScript. I practiced using loops to add event listeners and created a function that calculates a total based on user interaction. I also learned how to update the content of an HTML element using JavaScript, and how to apply a formatting function to display numeric values as currency.

Lab 3-3

In Lab 3-3, I learned how to use a for loop to iterate through an array and access data from multiple related arrays. I also learned how to insert a string into a webpage by setting the innerHTML of an element retrieved using its tag name.

Lab 3-5

In Lab 3-5, I learned how to identify and fix syntax errors in arrays and switch statements, logical errors in calculations, and problems related to calling and defining functions.

Lab 4-1

In Lab 4-1, I corrected logical errors in the cost calculations and added error handling to check that users enter valid input.

Lab 4-3

In Lab 4-3, I fixed some syntax errors with variables, added error handling to show a message if the character count goes over the limit, and cleaned up a mistake where the wrong variable name was used. I learned how to handle user input more safely and catch errors using try/catch blocks.

Lab 4-5

In Lab 4-5, I fixed syntax and logic errors and used console.log() to test and track variable values. This helped me understand how the code was working and made it easier to find issues. I also used the browserâ€™s developer tools to view error messages and figure out what was going wrong.

Lab 5-1

In Lab 5-1, I learned how to automatically run a function using the setInterval() method in order to update values in the HTML document. I also learned how to access all elements that match a specific CSS selector using the querySelectorAll() method.

Lab 5-3

In Lab 5-3, I learned how to get content from a web page and use JavaScript to add new things to it. I used a loop to go through heading elements and create links for a table of contents.

Lab 5-5

In Lab 5-5, I corrected errors in the scrambleTiles() and playConcentration() functions that were preventing the tiles from visually changing as expected when clicked.

Lab 6-1

In Lab 6-1, I added simple form validation with JavaScript to check that the password meets the pattern and matches the confirmation. I learned how to use setCustomValidity() to show custom error messages.

Lab 6-3

In Lab 6-3, I copied shipping info to billing and showed a custom error message if fields are left blank. I learned how to handle checkbox events and use the invalid event for form validation.

Lab 6-5

In Lab 6-5, I fixed errors by using proper event listeners and correcting element properties. I learned how to properly reference form fields and handle user input.
